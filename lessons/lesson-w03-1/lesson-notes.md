# Lesson-w03-1

## Primer

This is the week the block was built for. The diagnostic drew a sharp line: you can
NAME the participle constructions but not yet READ them — `litteras a Caesare missas`
inverted itself on you, and `urbe capta` turned into a conditional. The cause is
almost always the same: participles were learned as vocabulary, not as a system.
Today builds the system: four participles, all manufactured from principal parts you
already know, each with a tense-and-voice address. Nothing this week is new memory —
it's new wiring between things already in your head (4th principal part + agreement +
aspect).

## Lesson Plan

Target 30 minutes. One concept: the participle system — four cells, one machine.

1. **Hook (5 min).** English does this too: "the *falling* leaves" / "the *fallen*
   leaves" — what's the difference? (Aspect!) He already owns this distinction in
   Latin main verbs. Today: the same distinction, packed into adjectives.
2. **The 2×2 grid (12 min).** Build it WITH him, from capio:
   - present active: capiens (taking) — from present stem
   - perfect passive: captus (having-been-taken) — the 4th principal part, at last
   - future active: capturus (about-to-take) — 4th part + -urus
   - (future passive/gerundive capiendus: name it, park it — later block)
   Declension: captus = 2-1-2 like magnus; capiens = 3rd decl like ingens. He fills
   the grid for duco and sequor (deponent: secutus = "having followed" — ACTIVE
   meaning, the week-2 payoff).
3. **Tense is relative (8 min).** captus ≠ "past" absolutely — it's "before the main
   verb"; capiens = "at the same time as the main verb". Two examples with the same
   participle under different main-verb tenses. This principle will carry indirect
   statement in week 4 — say so.
4. **Wrap (5 min).** Form drill: he produces six participles on demand. Homework
   (~30 min): formation + parsing. Tomorrow: reading these in sentences — the
   diagnostic sentence itself will be there.

## Reference

Grid and paradigms in vocab-and-grammar.md. Planned examples:

- English hook: falling/fallen leaves; a *broken* promise vs. a *breaking* wave
- Grid builds: capio, duco, sequor (deponent twist)
- Relative-tense pair: `militem captum video` (I see the soldier who WAS captured) /
  `militem captum vidi` (I saw the soldier who HAD BEEN captured — before the seeing)
- Production drill: mitto→missus, venio→veniens, gero→gesturus, patior→passus,
  vinco→victus, scribo→scribens
