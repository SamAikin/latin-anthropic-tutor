# Lesson-w01-1 — Reference: Declensions 1–2 and the Case System

## The six cases and their core jobs

| Case | Core job | Signal question |
|---|---|---|
| Nominative | subject | who acts? |
| Genitive | possession / "of" | whose? of what? |
| Dative | indirect object / interest | to/for whom? |
| Accusative | direct object; goal with ad/in | whom/what acted on? |
| Ablative | means, manner, place, time; object of many preps | by/with/from/in? |
| Vocative | direct address | — (someone is being spoken to) |

## First declension — puella, puellae (f.)

| | sg. | pl. |
|---|---|---|
| nom. | puella | puellae |
| gen. | puellae | puellārum |
| dat. | puellae | puellīs |
| acc. | puellam | puellās |
| abl. | puellā | puellīs |
| voc. | puella | puellae |

Note: nom./abl. singular differ only by vowel length (puella / puellā) — in unmacroned
text (all our readings), context decides. Most decl-1 nouns are feminine; agricola,
nauta, poēta are masculine.

## Second declension masculine — dominus, dominī (m.)

| | sg. | pl. |
|---|---|---|
| nom. | dominus | dominī |
| gen. | dominī | dominōrum |
| dat. | dominō | dominīs |
| acc. | dominum | dominōs |
| abl. | dominō | dominīs |
| voc. | domine | dominī |

- -er nouns: puer, puerī (keeps e) vs. ager, agrī (drops e). The genitive tells you.
- -ius nouns: vocative in -ī (fīlī, not fīlie); genitive often -ī (fīlī) in classical texts.

## Second declension neuter — bellum, bellī (n.)

| | sg. | pl. |
|---|---|---|
| nom. | bellum | bella |
| gen. | bellī | bellōrum |
| dat. | bellō | bellīs |
| acc. | bellum | bella |
| abl. | bellō | bellīs |

**The neuter rule (all declensions, forever):** nominative = accusative; nominative/
accusative plural ends in -a.

## Ambiguity map (worth conscious attention)

- -ae = gen. sg., dat. sg., or nom./voc. pl. (decl 1)
- -ī = gen. sg. or nom./voc. pl. (decl 2 m.)
- -īs = dat. or abl. pl. (both declensions)
- -a = nom./voc. sg. (decl 1) or neut. nom./acc. pl. (decl 2)

Reading strategy: hold the ambiguity open until the verb and context close it.
