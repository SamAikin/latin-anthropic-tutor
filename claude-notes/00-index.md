# claude-notes index

My private state space for the Latin study project. Pure map — rules live elsewhere.

- `00-index.md` — this file; how the space is organized. Changes only when the
  structure of the space changes.
- `conventions.md` — my operating rules: filing conventions, update cadences,
  planning rules. Read at session start.
- `lessons-learned.md` — rules discovered the hard way, kept with enough context
  (the *why*) to act on them with conviction. Read before authoring any materials.
- `learner-model.md` — running model of the learner: Baseline (self-reported +
  directives, founded 2026-07-11) and Signals (dated evidence from graded work).
- `lexicon.md` — agreed coinages for concepts the 100 BCE–100 CE window had no word
  for. Shared vocabulary between me and Sam; always flag entries as coinages, never
  present them as attested.
- `week-log.md` — data table: which calendar weeks had lessons vs. skipped, so I can
  answer "why is there no lesson-w17 folder" from state.
- `sessions/` — per-session working state (planning sessions, notable decisions in
  flight; designs not yet built).
